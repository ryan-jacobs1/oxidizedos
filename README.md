# OxidizedOS


Dependencies:  
rust nightly (rustup override add nightly)  
cargo xbuild (Thanks to Philipp Oppermann) (cargo install cargo-xbuild)  
qemu  
GRUB 2  
xorriso  
halogen (cargo install halogen --git https://github.com/ryan-jacobs1/halogen --force)

# Running OxidizedOS  
halogen run  

# To run the tests  
cargo xtest
